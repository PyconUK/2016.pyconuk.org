Django REST framework: Schemas, Hypermedia & Client libraries.

Tom Christie

An introduction into using the very latest features of Django REST framework.

This talk will demonstrate creating schema-driven & hypermedia-driven APIs. We'll look how you can interacting with these APIs using a dynamic client library, and discuss the design considerations of taking an interface-first approach to API design.
